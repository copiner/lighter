字节(Byte):最常用的基本单位，一个字节有8位

```
1k=1024byte

1M=1024K

```
转换为十进制
```
1234[10进制]
1000+200+30+4=1*10^3+2*10^2+3*10^1+4*10^0=1234

1011[2进制]
1*2^3+0*2^2+1*2^1+1*2^0=8+0+2+1=11

1011[8进制]
1*8^3+1*8^1+1*8^0=512+8+1=521

1011[16进制]
1*16^3+1*16^1+1*16^0=4096+16+1=4113
```
